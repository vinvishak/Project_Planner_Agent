# Project Planner Agent

An AI driven agent that sits in your meetings, listens to conversations, and builds, tracks, and updates your project plan in real time.

## High level vision

- Capture meeting audio.
- Transcribe and understand discussion.
- Extract goals, tasks, blockers, and decisions.
- Maintain a living project plan with epics, stories, and tasks.
- Run lightweight standups and progress checks.

## Project layout

- `app/` source code for the agent, APIs, and core planning logic.
- `tests/` unit and integration tests.
- `docs/` design documents, architecture, and roadmap.
- `scripts/` helper scripts for local development and tooling.
